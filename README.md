操作票打印系统
这是一个用于生成和打印电气操作票的C# WinForms应用程序，支持将操作票导出为PDF格式。


功能特性
PDF导出功能：将操作票自动打印为PDF文件
多页打印支持：自动处理长操作票的分页
灵活的布局配置：可自定义边距、字体和表格样式
表格化显示：清晰的操作步骤表格布局
状态管理：完整的打印状态跟踪和页面管理


系统架构
主要类结构
核心打印类
PDF_write_PrintDocument_whj：打印入口类，负责打印机选择和PDF生成
打印_类：主打印协调器，协调状态管理、页面计算和绘制
打印_状态管理_类：管理打印进度、任务切换和页码计算
打印_页面计算_类：计算页面布局和组件尺寸
打印_页面绘制_类：执行实际绘图操作

布局组件类
页面环境_类：页面尺寸和边距环境信息
分页器_类：管理垂直分页和页面空间
分行器_类：管理水平单元格布局
计算行组件_类：计算各种行组件的尺寸和内容
行_类：表示页面中的一行，包含多个单元格
单元格_类：表格中的基本绘制单元

配置类
PrintConfig：集中管理所有打印配置（边距、字体、颜色等）


打印流程
选择PDF打印机
初始化打印文档和页面设置
计算页面布局
逐页绘制内容（变电站信息、操作任务、操作步骤表格等）
处理多页和分页逻辑
输出PDF文件


安装和配置
系统要求
.NET Framework 4.5或更高版本
Windows操作系统（使用System.Drawing进行打印）
PDF虚拟打印机（如"Microsoft Print to PDF"）
